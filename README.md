# Plant Care AI

Plant Care AI is a web application designed to help users monitor and maintain the health of their plants. The system provides guidance on plant care such as watering, sunlight requirements, and general maintenance through an interactive user interface.

The goal of this project is to combine modern web technologies with intelligent plant care assistance so that users can better understand the needs of their plants and maintain healthier growth.

---

## Project Features

* Smart plant care assistance
* Interactive user interface for monitoring plant conditions
* Guidance on watering, sunlight, and maintenance
* Fast and responsive web interface
* Modular and scalable project structure

## Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/plant-care-ai.git
```

### 2. Navigate to the project folder

```
cd plant-care-ai
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```

### 5. Open in the browser

```
http://localhost:8080
```
